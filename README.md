# 🌊 Tosa Quiz (高知・土佐検定)

高知県の文化、地理、グルメを楽しく学べる、モバイルファーストの3択クイズアプリです。

## 📝 概要
高知県にまつわる30問のデータベースから、ランダムに10問が出題されるクイズアプリです。
「高知県をもっと身近に」をコンセプトに、直感的で心地よい操作性を重視したシングルページアプリケーション（SPA）として構築しました。

## ✨ 特徴
- **ランダム出題システム**: 30問のストックから毎回異なる10問を抽出。
- **モバイルネイティブデザイン**: モダンなカードUI、Bento Grid、適切なタッチフィードバックを採用。
- **高知カラー**: カツオの赤（#e63946）と仁淀ブルー（#457b9d）を基調とした配色。
- **軽量動作**: ライブラリに依存せず、ブラウザ標準機能のみでサクサク動作。

## 🛠 技術スタック
- **Frontend**: HTML5, CSS3 (Flexbox/Grid), JavaScript (Vanilla JS)
- **Deployment**: GitHub Pages (推奨)

## 🚀 使い方
1. リポジトリをクローンまたはダウンロードします。
2. `index.html` をブラウザで開きます。
3. 全10問のクイズに回答し、最後にスコアと土佐弁のフィードバックが表示されます。
4. 「もう一度挑戦する」ボタンで繰り返し遊べます。
